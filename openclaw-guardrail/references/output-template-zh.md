# 中文报告模板（唯一输出模板）

> 所有最终报告必须按本模板输出；禁止混用英文。

## 标题区

```text
# 🏥 OpenClaw 安全体检报告

📅 {时间}
🖥️ OpenClaw {版本} · Node {版本} · {系统}
```

## 汇总表

```markdown
| 检查项 | 状态 | 详情 |
|--------|------|------|
| 配置审计 | {✅ 通过 / ⚠️ 风险} | {一句话} |
| Skill 风险 | {✅ 当前未见高风险 / ⚠️ 需关注 / 🔴 风险} | {一句话} |
| 版本漏洞 | {✅ 无 / ⚠️ 风险 / ℹ️ 情报暂不可用} | {一句话} |
| 隐私泄露风险 | {✅ 当前未见明显风险 / ⚠️ 需关注} | {一句话} |
| 综合评估 | **{✅ 当前未见明显高风险 / ⚠️ 需关注 / 🔴 风险}** | {一句话} |
```

## 风险明细表（有风险时输出）

```markdown
## 风险明细

| 级别 | 类型 | 位置 | 说明 |
|------|------|------|------|
| 🔴 | 凭证泄露 | src/config.ts:23 | 检测到硬编码密钥（已脱敏） |
| 🟠 | 供应链 | skill: foo-bar | 存在未声明外部下载行为 |
```

## 无风险时固定行

```text
✅ 当前未发现需要立即处置的高风险项，建议保持定期巡检。
```

## 报告尾部（必须出现在所有报告末尾）

无论是全面体检还是单 Skill 审计，报告最后一行必须是：

```text
如有安全相关问题，可联系：{从配置文件 policy.contacts 读取}
```

禁止省略此行，禁止硬编码联系人姓名（必须从配置动态读取）。
